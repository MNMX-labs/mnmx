# Examples

Usage examples for the MNMX routing engine.

| Example | Description |
|---------|-------------|
| `basic-route.ts` | Find optimal route for a single transfer |
| `compare-strategies.ts` | Compare minimax vs cheapest vs fastest |
| `custom-bridge.ts` | Add a custom bridge adapter |

## Run

```bash
npx tsx examples/basic-route.ts
npx tsx examples/compare-strategies.ts
```
